------- Comment #3 From Axel Grude 2019-03-01 06:52:45 ------- 
Can you enable debug mode and right-click it to also eenable the config setting

extensions.quickfolders.debug.css = true

Here is a description on how to do this and create a detailed error log:

http://quickfolders.org/support.html

in a nutshell:
1 - install console2, open with CCTRL+SHIFT+J
2 - configure to catch chrome + JS errors (disable content)
3 - clear output before the test

in order to see the error again, you most likely need to right-click a folder
and select "view in new window" - this should load the styleSheet code again.
If not, you hopefully will have the error log already on Thunderbird start.
