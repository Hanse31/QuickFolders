Thunderbird 3 compatible patch

Here is a patched verison (0.9 Prerelease 249). Took a while as I had to fix
some backwards incompatibilities introduced by Thunderbird's new AddonManager /
versioning Model. 

Also I had to test signatures / plain text quite extensively, hence the high
version number. If you are using %sig%, please make sure that you check the
attach signature options in Thunderbird Account Settings as well. 

Note: only English and German locale are complete! using other languages may
lead to errors.


Looking forward to your feedback.
