Description:   Opened: 2011-10-03 06:41 
After the last update (no version in the about) i've lost all my folders.
I can get the folder config via the settings dialog - but I can not restore the
settings.

Thanks

Bart
------- Comment #1 From Axel Grude 2011-10-03 08:53:43 ------- 
Generally I think a backup & restore function is a good idea, thinking of
something along these lines  (last known good folders would be also a nice
version of that) for QF 3.0.

The folder string can be restored manually to about:config, 

open Tools / Options / Advanced > General Tab and click on [Config editor]

set the filter to "QuickFolders.folders"  (without quotation marks)

Paste the string, click ok and restart thunderbird. Please note that you might
have bug 24451, which you need to fix before you can restore the folders.
Simply install version 2.9.1, it has been released today.

------- Comment #7 From Axel Grude 2019-06-03 11:52:03 ------- 
Created an attachment (id=8634) [details]
4.15.1 prerelease 88

This version deals better with invalid folder URIs which can happen when we
import a backup from an old profile (the folder was deleted or moved on the
profile in the meantime).

it will still load (and display) the tabs of non-existent folders but show them
with a "broom" icon and a tooltip to use "Find orphaned tab" to delete them.
They can be right-clicked for getting the QuiockFolders command menu. A normal
left-click on such an invalid tab offers to delete the tab.

note: this can also happen if a mail account has a different username when it
is set up on a different PC; the first pickeed username must match for the
folder URIs to be recognized on both systems. It is likely that renaming an
existing mail account will _NOT_ remedy this problem.
