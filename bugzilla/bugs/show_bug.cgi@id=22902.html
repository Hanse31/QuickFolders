------- Comment #5 From Axel Grude 2010-06-27 16:51:38 ------- 
Hi Michael,

seems to be some sort of chicken and egg situation here: if I include
quickfolders-interface.js before quickfolders.js I am getting "QuickFolders" is
undefined. it looks to me like OnItemIntPropertyChanged is triggered before
thunderbird gets a chance to load the quickfolders-interface file. Can you try
putting in a delay of 10000 in the "init delay" textbox on the first options
tab? Also you can try enabling "debug mode" and the debug setting "events" by
right clicking the debug box and then activating the
"extensions.quickfolders.debug.events" setting. If the same error happens again
it will be interesting for me to see whether it happens before or after
QF.init()

thanks
Axel
