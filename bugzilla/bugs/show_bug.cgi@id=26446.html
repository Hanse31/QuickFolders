Description:   Opened: 2017-11-18 10:25

with Task Manager when SmartTeemplate4 is enabled. There are many other XUL
based
addons that exhibit the same behaviors. See also Thunderbird  Bug:

https://bugzilla.mozilla.org/show_bug.cgi?id=1417209

I am working with the helpful guys at maildev (Joerg Knobloch etc.) to get this
resolved.
