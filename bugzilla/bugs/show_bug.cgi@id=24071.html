Description:   Opened: 2011-06-03 14:25 
The current folder toolbar is completely broken if messages are opened in a new
window.  None of the buttons work (except close, but the close button only
closes it temporarily).

To reproduce, either:
  * right click on a message and choose "Open Message in New Window"
  * go to Preferences..Advanced..Reading & Display and change "open messages
in" to "a new message window"
------- Comment #1 From Axel Grude 2011-06-03 15:23:55 ------- 
The obvious question is, should the toolbar be hidden completely, or should
navigation commands be added? I think the biggest advantage of the current
folder toolbar (apart from telling me at a glance in which folder I am) is that
it has a "this mail" icon, which can be used to drag the mail into a different
folder (or into a submenu, deeper within the folder I am currently in).

Any suggestions are welcome.
------- Comment #2 From Axel Grude 2011-06-04 02:51:48 ------- 
the new Current Folder toolbar was meant to be used in the message preview
window; I actually never use the "message in its own window" mode, so I missed
that during testing. I will need to find a way to make it not display in that
mode.


In preview mode "Current Folder" can be used to navigate folders and drag the
email to other folders, but it does not seem to work in the mode you have
described.
