Description:   Opened: 2014-11-14 16:08 
I would like an option to clear the "Apply Filter When" checkboxes as a default
setting for new filters.  Also, an option to not add a second condition
specifying the To: field with the same address as the From: field.  

I always have to uncheck the "Getting New Mail" checkbox, and delete the To:
condition.

I couldn't find anything in the configuration file that would do it, but if
there were, that would work as well.

Thanks!

Here is a test version. I had to reorganize the options dialog a little - the
option is on the second quickFilters tab 'Filter Properties':
[x] Apply Filter when getting new mail

just uncheck it before using quickFilters to generate a new filter.

note - ther versions real number is 2.8pre24 because I did not update the
version field in install.rdf
only English and German locale, so it currently won't work in other languages
of Thunderbird.

Tech stuff
----------

I removed the flags 
InboxRule 
InboxJavaScript
NewsRule  
NewsJavaScript

from the nsIMsgFilter.filterType attribute (this is a nsMsgFilterType)

The new filters should be at least tested on newsgroups and inboxes.
