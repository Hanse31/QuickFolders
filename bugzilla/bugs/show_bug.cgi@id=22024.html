Description:   Opened: 2009-11-16 18:32 
Emails seem to disappear when they are dragged from a folder (say inbox) to a
quick tab that points to a saved search "folder" and not to a real folder.
------- Comment #1 From Mark Edwood 2009-11-16 18:36:23 ------- 
p.s. Is there any way to find out where the emails went to? If so, is it
possible to recover them? I just noticed the bug, but I've been dragging emails
to the saved search folder for a while. I would like to recover those emails.
Thank you.
------- Comment #2 From Axel Grude 2009-11-17 01:53:43 ------- 
(In reply to comment #0)
> Emails seem to disappear when they are dragged from a folder (say inbox) to a
> quick tab that points to a saved search "folder" and not to a real folder.
> 

Yes, you can retrieve the emails:

it seems like Thunderbird simply creates a folder of the name of your search.

navigate to your mail profile
(e.g. in Windows)
C:\Documents and Settings\<your username>\Application
Data\Thunderbird\Profiles\<profile name>\Mail

and search for the name of the the "search folder". You will find 2 files, use
the one without extension and copy it to your inbox. Restart THunderbird - you
can now access the "nirvanaed" emails.

I must see if I can fix this by preventing folder operations on search folders.
Should they even appear on the QF toolbar? There are already some bugs against
search folders (such as highlighting not working) so I wonder if it is worth
the hassle?

No idea, make sure you only copy the file without the extension - or maybe try
putting it into a different location - I don't really know what Thunderbird's
strategy is in picking a "first location" for a search folder...obviously
moving real mail into it and then moving it elsewhere will treat it like normal
