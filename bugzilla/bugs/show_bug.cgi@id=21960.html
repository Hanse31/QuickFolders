------- Comment #4 From Axel Grude 2009-11-02 11:02:47 ------- 
Can you check whether the icon file is there first of all:

In windows it would be here:
C:\Documents and Settings\<username>\Application
Data\Thunderbird\Profiles\<profilename>\extensions\quickfolders@curious.be\chrome\content\folder-inbox.png

(don't know where the profile is on the Mac, sorry)

Second, can you try to go to the options page and click the [Folders?] button -
it should copy the folder string to your clipboard. 

you should find an entry something like this

[{"uri":"mailbox://<your email address>/Inbox","name":"Inbox",
"category":"__ALWAYS",
"tabColor":"1"}

The icon is added at runtime by matching the URI to end with the string /Inbox.
Could it be that it has a different name in your system (either because it is
not the real Inbox or because its called different on the apple OS?

Looking forward to see what you may find.

Axel.
