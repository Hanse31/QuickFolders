Screenshot of the tooltip

When using 
<a title="%from%">%from(name)%</a>

in a header template for "reply" or "forward",

then I get in the email the tooltip <a>mail@ddress.abc</a> instead of the
tooltip mail@ddress.abc
------- Comment #1 From Axel Grude 2013-10-16 06:52:13 ------- 
try this instead:
%from(name,link)%
------- Comment #2 From David.P 2013-10-16 06:59:33 ------- 
Thanks Axel, that works.

is there also a way to have a tooltip on the name link (that shows the mail
address)?
------- Comment #3 From Axel Grude 2015-07-09 04:27:27 ------- 
(In reply to comment #2)
> Thanks Axel, that works.
> 
> is there also a way to have a tooltip on the name link (that shows the mail
> address)?
> 

I am not sure what you are asking, what is the HTML markup you are truying to
achieve. Please also chek out the new bracketMail() and bracketName() functions
which can be passed as part of the %from% variable:

e.g.

%from(bracketName(";"),link)%

Maybe link could be extended in a similar fashion with an argument, such as

link([tooltip])

Also it might be useful to have possibility to link to a web address, with
something like this:

linkHtml([URL],tooltip)
