Description:   Opened: 2012-08-17 02:42 
%to(name)% should fall back to %to(email)% if sender is not in Address book
(and Name cannot be safely determined from the Email Address)
------- Comment #1 From Axel Grude 2013-05-29 02:55:47 ------- 
1. st4 does not currently support address book lookup; maybe you should request
that in a separate bug, because it would be more useful than this one, and have
more chance to be implemented first

2. There is no standard way of including the senders name in the email address
field, so st4 uses some fairly basic assumptions on where and how the name is
included (if it is):

No name
  xx.yy@zzz.com

xx = First
yy = Last

Name somwhere:
  xxxx<email@zzz.com>

removes the portion between the < >

   xxxxx
   xxx.yyyy
   xxx,yyyy
   xxx yyyy

xxx = first name
yyyy = last name(s)

if the new "swap first,last" is set, then they are swapped in the comma case

I will attach the source code to the "split" routine for full disclosure.
