------- Comment #2 From Axel Grude 2011-05-22 02:25:26 ------- 
I have found out that popupNode is deprecated in future versions of Gecko (it
still shows up in Postbox2.5 but it is set to null). It is replaced in favor of
the new triggernode attribute which appears to belong to the menupopup owning
the calling menuitem. Unfortunately this also is null in the Postbox beta, I
don't know whether this is a bug in the gecko codebase or in Postbox itself.

For the meantime, I have coded a {somewhat clunky} workaround that traverses
the dom tree up to the toolbarbutton, will attach it to the bug.

For more information on this, please refer to:

https://bugzilla.mozilla.org/show_bug.cgi?id=383930
