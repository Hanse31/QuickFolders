------- Comment #8 From Axel Grude 2015-07-09 04:43:35 ------- 
To consolidate, and send users to a possible solution at least for local files,
I am marking this one as duplicate of [Bug 25871] which deals with external
files.

Usage - HTML files:
%file(localpath,encoding)%

Usage - Images:
%file(localpath,altText)%


use the variables tab to insert the variable to be prompted with a file picker.

Maybe we can eventually support loading these files from a server
asynchronously. But it is very complicated.
