Description:   Opened: 2010-06-06 09:56 
Hi,

When the option to sort alphabetically menus with subfolders is enabled, the
folder itself is part of the sort of the subfolders' list displayed.

It would be great to have the possibility to keep the folder aside this sort,
leaving it displayed as the first occurence.

Beside, great add-on!

Regards,
Dji
Using QuickFolders 1.9.1 on Thunderbird 2.0.0.24, Win XP SP2

When the sorting by default is enabled, QF doesn't show a correct order of the
items (see Attachment #1 [details]) with accentuated characters when Thunderbird's tree
display is correct.
In attachement #1, for instance, "éditions du café" should appear in between
"correspondance par email" and "fabbian".

For that reason, I choose to force the alphabetical sorting of menus with
subfolders.

Unfortunatly, taking that option changes the positionning of the menu name
within the subfolder's list (see attachment #3 [details]).
It would be great if it were possible to keep the menu name at the first
position of the subfolder's list as it is the case without the alphabetical
sorting (as shows the attachment #2 [details]).

Hope I'm clearer !

Rgds,
Dji

The option  'force alphabetic sorting of menus...' was designed to address this
- are there any accents/diacritics that I missed? You can send me an (html)
email (use the Support / Contact Me Option) with the ones that are missing (as
I do not think you can post all special characters on the bug pages), as I only
replace a limited number of special characters in my alphabetic sort algorithm.

this is the list of characters I currently support
var tr = {"\xE0":"a", "\xE1":"a", "\xE2":"a", "\xE3":"a", "\xE4":"ae",
"\xE5":"ae", "\xE6":"a",
"\xE8":"e", "\xE9":"e", "\xEA":"e", "\xEB":"e",
"\xF2":"o", "\xF3":"o", "\xF4":"o", "\xF5":"o", "\xF6":"oe",
"\xEC":"i", "\xED":"i", "\xEE":"i", "\xEF":"i",
"\xF9":"u", "\xFA":"u", "\xFB":"u", "\xFC":"ue", "\xFF":"y",
"\xDF":"ss", "_":"/", ":":"."};

that's these ones (don't know whether they will come out correct):
à, á, â, ä, å, æ, 
è, é, ê, ë,
ò, ó, ô, õ, ö,
ì, í, î, ï,
ù, ú, û, ü, ÿ, ß

You can also use
http://rishida.net/tools/conversion/
for turning characters into UNICODE hex codes.
