Description:   Opened: 2014-11-21 08:36 
When the template is added to your message, it would be nice if we can specify
where we want the cursor to be initially.

I use this add-on to apply different signatures for new messages versus replies
and forwards (new messages have a disclaimer while replies and forwards don't).
 But when the template is applied to the message, the cursor is placed AFTER
the signature, forcing me to always move the cursor to the top of the message
before I can start typing my reply.

Although I'm submitting this as a feature request, I'm not sure if this is a
bug or intentional since it seems the "Number of line breaks at top" setting
should add the lines above the template so that if the cursor actually did
appear above the template, you can start typing with enough space to work with.
 But it actually adds the lines AFTER the template/signature, forcing me to
manually add the lines at the beginning of the template instead AND then
manually moving my cursor to the top.
