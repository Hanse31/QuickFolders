Description:   Opened: 2014-02-26 03:25 
Right now if I select "Based on Sender (from)" it assumes that I want to have
both a From and To filter in the new filter.  I believe this is a bad
assumption that does not apply to all users.  I appreciate your consideration
for this request.
------- Comment #1 From Axel Grude 2014-02-26 08:57:56 ------- 
I can add this as additional global option (which will be disabled as default):

[x] disable two-way address search terms

tooltip: By default, quickFilters fills 'from' and 'to' search conditions to
aid filtering message threads including your own emails to the recipient. The
Addon 'Copy Sent to Current' can be used to keep replies with their original
emails. If you always want to keep your replies in the Sent folder, activate
this option.

I will probably also add a link to:
https://addons.mozilla.org/en-US/thunderbird/addon/copy-sent-to-current/

to make the use of two way filtering more obvious
