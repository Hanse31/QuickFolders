------- Comment #5 From Axel Grude 2018-04-11 08:26:03 ------- 
try this:

%matchTextFromSubject("(?:\S+\s+){4}(\S+)",1)%

the first group argument will be repeated 4 times
(?:\S+\s+)

the second group argument (index=1) returns aaa-bbb
