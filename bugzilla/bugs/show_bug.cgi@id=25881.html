How about a checkbox on the Mail Account / Server Settings page:

Underneath the check mail timer we could have an option:

[x] Check for new messages every [...] minutes
[x] Run Filter on existing emails after check

this could automatically run the filters when
a) mails are downloaded automatically
b) every time the user clicks the "Get Mail" button.

Since the settings are account specific, so configuration would give more
control.
