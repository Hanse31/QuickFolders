Description:   Opened: 2018-11-07 03:29 
I suggest that quickFilters gets the ability to save attachments from emails as
a custom template. Currently attachmentExtractor and FiltaQuilla are able to
perform this function but have not been updated to be compatible with
Thunderbird 60.*.
------- Comment #1 From Axel Grude 2018-11-07 04:13:51 ------- 
FiltaQuilla is really the best fit for this (as it adds custom actions, which
is not something quickFilters currently does; for the moment I want to keep
that functionality separate for the smaller number of users with advanced
requirements)

I have reached out to Kent James (owner of filtaQuilla) and am currently
organising the adoption of the Add-on. Once I have full ownership we can file a
separate bug for it to track progress. 

Until then, I will keep you updated here.
