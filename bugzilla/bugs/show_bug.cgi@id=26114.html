Description:   Opened: 2015-12-05 09:08 
As soon as the selection changes the context menu items

"Copy Site URI"
"Copy selected lines to the Clipboard"
"Unified Password change"

are disabled. This is caused by soem code in Firefox itself and makes it
impossible to highlight and export all data from the password manager.
------- Comment #1 From Axel Grude 2015-12-05 09:29:00 ------- 
Created an attachment (id=8133) [details]
3.6 prerelease 96

This version will enable the following menu items depending to whether one or
multiple items are selected:

(1) One item selected - what is ENABLED:
Login to Website
Copy User Name to Clipboard
Copy Password to Clipboard
Copy Site URI
Copy selected lines to the Clipboard (will copy password data if visible)
QuickPasswords Options
Unified password change...


(2) Multiple lines selected - what is ENABLED:
Copy selected lines to the Clipboard (will copy password data if visible)
QuickPasswords Options

The other menuitems remain disabled because for this case they are ambiguous.
