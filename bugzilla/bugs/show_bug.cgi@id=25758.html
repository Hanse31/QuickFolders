Created an attachment (id=7857) [details]
The error message

Initial installation of add-on was unable to create any filters, regardless of
the method attempted. Every attempt created message as shown in attached screen
shot.

Problem turned out to be caused by an over-aggressive sanity test which
triggers the exception if any account in the system has an empty email address.
I found one account (unused leftover from initial learning curve for TBird) in
this condition.

Removing that account entirely solved the problem.

Just to clarify in your case the application shows an error message:

and no reply to address. I will post a bugfix shortly.
