Description:   Opened: 2012-08-24 09:23 
I have a longer signature for work with multiple lines in it (address, phone,
fax, etc.).  I use the settings to have my reply above the quote, but my
signature below the quote.  Additionally the signature is entered into TB in
the account settings page (not in ST4).  Since version 0.9.1, it turns it into
one long line.

Example as shown in 0.8.5.6 version:
Jon Doe
123 45th Street
City, ST 12345
Phone: (123)456-7890

Now shows as:
Jon Doe 123 45th Street City, ST 12345 Phone: (123)456-7890

I can install the older version to keep it working and have turned off
automatic updates on this extension for now.

------- Comment #2 From nischwartz@yahoo.com 2012-08-24 14:38:03 ------- 
I hope this is all the info you need:

TB 14.0 Settings:
    Under Account Settings:
       Signature text: Use HTML is not checked
       Signature text is written into box below without any tags/formatting 
    Under Composition & Addressing:
       Compose messages in HTML format is checked
       Automatically quote... is checked
       Then, "start my reply above the quote" and place my signature "below the
quote (recommended)"
       Include signature for replies is checked
       Include signature for forwards is checked

ST4 Settings:
    Apply the following template... is checked for Write, Reply and Forward
    Use HTML... is not checked for all

NOTE: After changing signature to HTML and adding the <br> codes it is working
properly in version 0.9.1

Thanks for the fix, but I'm sure that there a many people who would still need
it to work without having to select HTML and add the <br> tags.
