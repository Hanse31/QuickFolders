Description:   Opened: 2010-07-30 08:55 
The Active, Hovered and Drag tabbed colors specified in the options only appear
to work in the striped style version for colored tabbed. It would be helpful if
these colors could override the filled style colored tabs as well as the
indication, especially for the active tab, is very subtle without a different
color being applied.
------- Comment #1 From Axel Grude 2010-08-15 13:55:52 ------- 
Created an attachment (id=6555) [details]
A version that changes active / hovered styles to show colors

The attached version 1.9.6 prerelease 3 attempts to fix the bug by dynamically
changing active and hovered tabs to "striped" style when the "filled style" is
enabled. This will make the tab stick out more without removing the users color
choice completely from the tab. 

Known issues: Note that the font color currently does not change during hover
which can lead to a temporarily unreadable label. Doesn't work for dragging
emails, yet.
------- Comment #2 From Axel Grude 2010-08-16 06:07:06 ------- 
Some more styles added in Release 1.9.7 - this should close the bug, for now. 

New behavior:
1. Active Tab: shows striped stile with active background color (an animation /
glow might be better, because its still a little confusing)
2. Hovered Tab: hovered color replaces the customized tab color
3. Dragged Tab over Tab: drag color replaces the customized tab color; arrow
indicator is displayed as before.
4. Dragging Email over Tab: drag color replaces the customized tab color.
5. Dragging Email over Active Tab: Currently, no change.

Of course there are still improvements possible... the CSS is a little
inefficient and might require a rewrite to speed up and use up less resources.
------- Comment #3 From Magritte 2010-08-16 11:01:20 ------- 
This is a big improvement. Thanks for fixing this! The only thing I'd say could
be improved (at least visually) is changing the font color on hovered/dragged
tabs. But this isn't such a big deal as, unlike the active tab, hovering is a
temporary state and you can always quickly move the cursor off the destination
tab to improve readability if necessary.
