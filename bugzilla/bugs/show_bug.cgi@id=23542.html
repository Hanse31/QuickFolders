Description:   Opened: 2010-12-17 11:57 
I am now trying to use QuickFolders 2.2 with Postbox 2.1.0 under Mac OS X
10.6.5.
I can create 'Quick' folders but dragging email messages from my inbox (for
example) to one of the folders located within a QuickFolder tab does not move
the email to the desired folder.  The email remains in the original location
(in this case, the InBox).
------- Comment #1 From Axel Grude 2010-12-17 16:43:12 ------- 
Can you please check Tools / Error Console if there are any relevant error
outputs? MOving mail both to tabs and subfolder works for me under Windows XP
SP3 (Postbox 2.1.0).

If there is no message, you can also try to enable Debug Mode and (via right
click on Debug) the options

extensions.quickfolders.debug.dnd
extensions.quickfolders.debug.events
extensions.quickfolders.debug.mouseclicks

You should get at least these 3 messages (and some others that deal with the
menu display in between) :

buttonDragObserver.onDrop flavor=text/x-moz-message

QuickFolders.Util.moveMessages: target = <your folder name>, makeCopy=false

event: DeleteOrMoveMsgCompleted
