Description:   Opened: 2012-09-29 14:02 
+++ This bug was initially created as a clone of Bug #25084 +++

My "Reply" template generates a blank line AFTER my actual template text. The
template contains:

--------------Original Message--------------
Subject: %subject%
Date: %date%
From: %from%
To: %to%[[
CC: %cc%]]

and there is no CR LF after the "CC:" line.  0.9.1 pre-release 2 does not
address this issue.

Here is a sample reply:

--------------Original Message--------------
Subject: Re: Visual Studio recent service pack
Date: Sat, 29 Sep 2012 16:20:26 -0400
From: so-and-so
To: me


So I'll meet you at 10am

----------------------------------------------

Note the extra line after the "To:" and before the "So I'll meet you at 10am". 
Also, if there was a "CC:", the extra line is there too, so it has nothing to
do with the conditional CC.
