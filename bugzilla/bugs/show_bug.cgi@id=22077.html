Description:   Opened: 2009-11-30 14:02 
After an autoamtic Quickfolders Upgrade to version 1.6.4, at least two
previously defined tabs became invisible in the selection list (but when trying
to add them as tabs I get a message telling me they are already bookmarked). 
And with the 'Display All' selection only one of the defined tabs is shown.  

This occurred on Icedove 2.0.0.22 (in Debian Lenny).
Other installed add-ons: 
- Display Mail User Agent 1.6.0
- Enigmail 0.95.0
- FolderFlags 1.1
- Nostalgy 0.2.20
- Remove Duplicate Messages 0.3.2
- SmoothWheeel 0.44.19.20090811.3

Hi J M,

I will be contacting you directly with some screen shots, it should be
relatively easily to solve - I bet I can fix it quickly:

with the new version 1.8.3 (I have attached it* ) - go to the Options / General
page. Right click on the Debug checkbox  - about:config will come up.

If there is a warning "this will void your warranty" or similar just check the
box and click Ok.

Now toggle events + folders so their value reads true. the first setting
(debug) should already be set to true if you have checked the Debug box. Close
the window.

Also you should set the Options Display Icons + "Display Tabs as Flat Style" on
the second tab of the QF Options dialog.

Once you're done, send me the debug output - just to make that real easy for
you I highly recommend using console2:

The most comfortable way to do the error console is to use console2 - its
always the very first extension I install on any Thunderbird installation as it
is so handy for troubleshooting.

Console2  ::  https://addons.mozilla.org/en-US/thunderbird/addon/1815

Once installed, go to Tools / Error Console - click All and then unselect
"Content".

Then you can just highlight a range of messages and copy them to the clipboard.
Since you're using folder flags it will be interesting to see whether the Junk
and Inbox Icons do appear when you have enabled the "Display Icons" option.

Do post the debug output to me directly to avoid showing the email addresses
here; I guarantee everything is treated confidential.
