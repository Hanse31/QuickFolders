Description:   Opened: 2018-02-27 00:30 
My template looks like this

-------- Forwarded Message --------
Subject: %subject%
Date: %date%
From: %from(mail)%
To: %to(mail)%
[[Cc: %cc(mail)%]]


When the Cc line is short, there is an empty line added between the message
body and the header, as intended. When there are more recipients and thus the
Cc line spans multiple lines, that empty line is omitted, adding the message
body directly unter the header.
------- Comment #1 From Axel Grude 2018-02-27 08:58:16 ------- 
You can start by switching off that pesky option "Replace line breaks with
<br>" and putting this into the

"Quote Header" (not "Template")

field:


-------- Forwarded Message --------<br>
Subject: %subject%<br>
Date: %date%<br>
From: %from(mail)%<br>
To: %to(mail)%<br>
[[Cc: %cc(mail)%<br>]]

then see if the problem still persists.
