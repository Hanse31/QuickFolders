------- Comment #3 From Axel Grude 2009-07-12 00:53:59 ------- 
Thanks for reporting - somebody reported this bug well before QuickFolders was
added to Bugzilla. At that time I did not manage to resolve it, but I will look
into it again - which version of TB is relevant for this one?

As a workaround, you can add the following rule to userchrome.css:

menupopup.QuickFolders-folder-popup menuitem {
  line-height: 4px !important;
  max-height: 14px !important;
  margin: 0px !important;
  padding: 0px !important;
}

you can also add a rule for font-size as well. This might remove the need for
scrolling.
