Description:   Opened: 2010-09-22 08:59 
Hi,

TB 3.x supports Tabs by now. In the folder pane you can open a folder as a new
tab in TB.

I'm working with Nostalgy and hide the folder pane most of the time.
Therefore I always have to show the folder pane, select a folder for a new tab
and hide the folder pane again.

What do you think about the feature of opening a new tab of a folder from the
context menu of a QF tab?

Greetings
korinor
------- Comment #1 From Axel Grude 2010-09-22 12:55:27 ------- 
Hi Korinor, of course it is possible to add that as a context menu item, but
the feature as such already exists: 

Hold down CTRL while clicking on a QF tab, this will open a new mail tab in
Thunderbird.

Axel
------- Comment #2 From korinor 2010-09-22 22:09:01 -------
