Description:   Opened: 2011-06-24 08:38 
Thunderbird 2.0.0.24 on Ubuntu. Invited to upgrade to 2.7 today (24/6/2011).
All folder tabs disappear, and ALT+1, ALT+2 etc shortcuts fail to work (nothing
happens). Attempting to re-drag the folders to the QuickFolders bar reports
that the folder is already present. ALT-left and ALT-right work as normal.

I tried a few things, but the breakthrough came by renaming the file from

to

I do not quite understand why TB2 has a problem here. as quite a few of the JS
script files also have "-" characters in them. Just to be safe I also renamed
the other XUL files that contain "-" characters.

The attachment is a patched preview version, this patch will be included in the
next version 2.7.1
