------- Comment #6 From Axel Grude 2018-01-22 02:25:51 ------- 
After having thought it over and thinking on how it may be valuable but also be
better integrated into QuickFolders (by adding an option to create a QF tab)
here is a suggestion on how to implement (Deutsche Übersetzung darunter):

A configurable toolbar button which opens a Inputbox style dialog.

Dialog Description
==================

Text field: Enter Folder name
Checkbox: [x] create QuickFolder Tab
Dropdown to select QuickFOlders Category.
Buttons: [Ok], [Cancel]

The dropdown is only shown if Categories are defined. (Happens automatically as
soon as a QuickFolder is sorted into a new Category)

If shown, the current Category is preselected.
If shown, the category dropdown is initially disabled.
Once the checkbox "create QF Tab" is selected the drop down becomes enabled.

GERMAN TRANSLATION

Eine optionale Schaltfläche die einen Dialog mit Eingabefeld 

Eingabefeld: Verzeichnisnam
Checkbox: [x] QF Tab erzeugen
Dropdown Menue fuer QF Kategorie
Schaltflaechen: [Ok] [Abbrechen]

Das menue wird nur gezeigt wenn QF Kategorien definiert sind und ist anfangs
gesperrt, aktuelle Kategorie ist ausgewaehlt. durch aktivieren der checkbox
wird die dropdown freigegeben.
