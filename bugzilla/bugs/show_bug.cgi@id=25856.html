Description:   Opened: 2014-09-24 16:10 
"Display the total number of messages on the folder button" would be very
helpful to me if it was set for each tab individually.
This because some tabs/folders will never be empty but on others it's useful to
see how many there is left without opening.
------- Comment #1 From Axel Grude 2014-09-25 02:18:22 ------- 
Just a heads up - thanks also for contacting me directly, Since this is no
doubt an "esoteric" function that only a small number of users would need, I
need to think through how to make this possible without further cluttering the
QuickFolders commands menu.

I think the best thing would be to bundle all "tab-specific" customizations
like this into one menu item "advanced behavior" or something which would open
a dialog and would be indicated by a checkmark to show that something
"non-standard" is going on with that folder. This way we can extend his with
various "exceptions" of the kind you suggested later on without adding more and
more menu items.

Currently the QuickFoilders Commands menu has the following items:

Tab Color (Popup)
Set Category...
Remove Tab
Rename Tab...
Customize Icon... (from version 3.15)
Remove Customized Icon (optional, only shown if icon was customized)
Insert line Break  (this is a toggle)
Insert Separator   (this is a toggle)

I suggest the following new structure

Tab Color (Popup)
Set Category...
Remove Tab
Rename Tab...
Insert line Break  (this is a toggle)
Insert Separator   (this is a toggle)
-------separator -----------
Customize Icon... (from version 3.15)
Remove Customized Icon (optional, only shown if icon was customized)
Advanced Properties...  (with checkbox)

------- Comment #4 From Axel Grude 2014-09-26 08:46:09 ------- 
Created an attachment (id=7925) [details]
3.15 prerelease 343

Here is a test version - it "does what you want", but note that the setting is
currently not stored, so it is just purely to try out the behavior.

The new "Advanced Properties" dialog has 2 checkboxes

"Ignore Unread Status"
this removes the "Bold" status and also the (Count) for unread emails

"Hide Count of Mails"
this can be used to also hide counts for all other mails within the folder (if
the "Show counts" global settings is active)

There is also a display panel for Categories and [...] button to set them
(which can also be opened via the "Set Tab Category..." menuitem.

Reset should restore default behavior.

Note: The UI is currently only refreshed when the [Ok] button is clicked; this
might have to change for Linux / Mac users.

3.15 prerelease 364

New version, with a bit more "ZING!" added. This one stores the advanced
settings, but at this stage please use at own risk (in previous builds my
folder settings were deleted a few times). For testing, I highly recommend to
make a backup of the folder string, using QuickFolders Options > Advanced and
click the [Folders?] button at the bottom. snatch the folder string from the
clipboard and store it in a file. It can be then restored with the Config
Editor via the setting "QuickFolders.folders"...

I added an option for custom coloring as well (including some links for
appropriate CSS syntax), and the checkbox on the advanced menu now indicates
that a flag is set.

The [Reset to Default] button closes the dialog as well for ease of use.
