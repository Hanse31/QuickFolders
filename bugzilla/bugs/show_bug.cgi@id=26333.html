Description:   Opened: 2017-01-31 10:28 
it appears firefox does not recognized the redbox login site for saving login
info.  after entering the site login info, there's no dialog box to save the
login info.  subsequent launches and site visits require re-entering the login
info because the username and password are never saved.  something about this
site doesnt trigger the save login prompt.
------- Comment #1 From Axel Grude 2017-01-31 12:13:29 ------- 
I can't even go on the main site (getting "Access Denied"). If you want to
force Firefox to save a login, please install "Saved Password Editor" and then
use the "New Login" button (to the right of [remove All] and if no web from is
found try the "Annotated" function. I highly recommend using SPE alongside
QuickPasswords and made sure I wouldn't replicate any of its functionality in
my own Addon.

One problem with somee sites is that they don't use forms (and may use flash)
for asking for passwords. QuickPasswords may be able to help filling them but
you have to make sure Firefox has them stored first.
