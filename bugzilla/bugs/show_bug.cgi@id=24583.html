Description:   Opened: 2011-11-21 07:21 
Axel,

FEATURE REQUEST: I need a "last folder" button that will switch me instantly to
the previously viewed folder (just like the 'previous channel' button on a tv
remote control), or even toggle between the last three viewed folders.

JUSTIFICATION: I do not use folder tabs, but I often search for a message
between two or three folders. I need to jump quickly between two three folders.
Now I have to right-click my QuickFolders folder tab and then select the
subfolder I need. This can be a bit tedious when I need to jump back and forth
between two subfolders under different main folders. Outlook has a 'go back'
and 'go forward' button. I searched in vain for a toolbar button or extension
that would do this in Thunderbird. I know that if YOU make one, Axel, it will
work flawlessly! :-))

Thanks,
Alan
------- Comment #1 From Axel Grude 2011-11-21 08:19:35 ------- 
I know what you want, I don't quite know what you mean though. Since it is
added to the QuickFolders bug list, (but with the provision "I don't use folder
tabs"??) would you want this as feature of QF? In that case I would recommend
using the Recent Folder dropdown (QF bar, left hand side), just drop it down
and the last touched / viewed / "mail moved to" Folder is on the top. 

However if you want to use this on a system that hasn't got QuickFolders
installed, why not use Back and Forward buttons? You merely have to drag them
to the toolbar, I always use them to find my way back into Folders that I just
visited.

BTW the current folder toolbar also shows my recent folders dropdown, and the
back and forward buttons on the left hand side.

------- Comment #5 From Axel Grude 2011-12-05 00:10:27 ------- 
(In reply to comment #4)

> I'm not sure how to use the Error Console- it's Greek to me!

Install the console2 extension and switch on [All] errors. then disable
[Content]. anything that is red or yellow (Errors, Warnings) is of interest.
With console2 you can right-click and copy the errors as text - if you get any
output whatsoever when clicking the TB navigation buttons then this might be
helpful.
