Description:   Opened: 2011-06-03 00:41 
Changes after upgrading from 2.4 to 2.5:
* Tabs *without* individual color setting lost transparency.
  (Not sure I'd call that a bug - it obscures the persona (just like
  Firefox tabs do ...), but also makes the tab title easier to read ...)
* Tabs *with* individual color setting don't show the active/hovered/
  dragged color change anymore. (Now *that*'s plain confusing. ;-)

Platform: Thunderbird 3.1.10 (with a total of 13 active add-ons ...)
