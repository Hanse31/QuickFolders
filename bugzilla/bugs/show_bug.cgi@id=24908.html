Prototype: Palette Editor

This is a mockup for a palette editor, which supports user generated gradients
with up to 7 color stops and transparency. It can use the standard color picker
or a derivative of rainbowPicker to define the color stops. A css string edit /
import would be another good idea, so that people can use external editors such
as http://www.colorzilla.com/gradient-editor/ to generate the gradients.
