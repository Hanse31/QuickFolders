Description:   Opened: 2012-08-04 06:47 
QuickFolders 3.6

When dragging a message to a tab it usually doesn't highlight the tab but
rather drops down a "New Subfolder" menu.

Please add a new menu configuration option to disable this behavior.

Thank you
------- Comment #1 From Axel Grude 2012-08-04 08:12:16 ------- 
Good suggestion - I can disable dropping down the menu (provided there are no
sub-folders within the folder, as these are always expanded into the dropdown
menu). 

However, highlighting the tab is already working, there is a dragover effect
that should be visible (and different from the normal hover effect) - this can
be set in "bling my tabs!" > Colors > Drag Over Color
------- Comment #2 From Axel Grude 2012-08-04 09:33:07 ------- 
Created an attachment (id=7331) [details]
test version - 3.7.1 Prerelease 16

Here is a test version - only has English and German locale complete (so the
options dialog will crash if used in any other language). Took a bit of time as
I had to guard against an empty popup menu (the rules for building menus are
rather complicated, so this is not as trivial as it may sound).
