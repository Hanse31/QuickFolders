------- Comment #1 From Axel Grude 2012-11-15 16:08:54 ------- 
Created an attachment (id=7416) [details]
3.8.2 prerelease 3

This has a hidden option for showing the complete path, but there is still a
problem with the menu width causing the path to be cropped at the end.

To set full path in description, go to QuickFolders General Options tab and
right-click on the "Display Recent Folder Tab" option in order to get advanced
options about the recent folders. then find the setting

extensions.quickfolders.recentfolders.folderPathDetail

and set as follows:

0 - folder name
1 - folder name - account name
2 - full folder path
