------- Comment #2 From Axel Grude 2019-01-02 10:44:26 ------- 
Created an attachment (id=8575) [details]
2.1 pre 17

This version allows using the variables from [Bug 24993] and also %sig()% 
within a Thunderbird template.

Supported variables:

%subject%
%from%
%to% %cc% %bcc%
%identity%
%date%
%dateformat()%
%sig%

If %sig% is omitted then it should be treated according to the Thunderbird
Composer account settings. To force omission of signature, add %sig(none)%. To
determine a specific position *within* the template, use %sig(html)%.

The variables from above (except %sig% itself) should also usable within the
signature HTML.
