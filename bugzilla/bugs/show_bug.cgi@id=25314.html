Description:   Opened: 2013-01-21 14:28 
Running QuickFolders 3.10 on Thunderbird 17.0.2 on Ubuntu 12.04.

I've noticed when hovering from tab to tab there is an animation that causes
the background color to fade from the default to the hover color. The animation
feels a bit slow, and makes the UI feel responsive than it actually is. I don't
think it is due to performance, but the timing values chosen.

Is there an option to disable the animation?

Let me revise the ticket then:

Rename "Enable CSS transitions" to something more obvious. CSS is an
implementation detail that will only make sense to technical users. Something
mentioning tabs, animation, or fading would be good.

Additionally, I unchecked the box, and there was no change. (Though some of the
black border (tab bar background color) below the tabs disappeared.) I tried
the refresh option. Background colors still fade in on mouse over.

Do I need to restart TB for the change to take effect?
