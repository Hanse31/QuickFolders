------- Comment #5 From Markus Stadler 2014-04-19 08:44:15 ------- 
Probably it is releated to the setting

browser.preferences.instantApply=false (which is the default for Windows)
browser.preferences.instantApply=true (which is the default for Macs/Linux)

which is only activated under MacOS but not under Windows.

For reference see this bug: https://bugzilla.mozilla.org/show_bug.cgi?id=641288

I test now your 3.2.1 prerelease 16.
