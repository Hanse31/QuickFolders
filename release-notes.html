Version 4.15.1
  • [Bug 24468] Added backup and restore functions on the QuickFolders Pro Tab. Save Configuration will store everything including all QuickFolders Tabs, general settings, layout and advanced settings.The later includes advanced settings except for the license key. Load Configuration will only load whatever option is selected.
  • [Bug 26670]
  • Improved the behavior of the "Find orphaned Tabs" command. This now includes an option to reuse the answer (delete remaining faulty tabs - Yes / No) rather than clicking once for each invalid tab.
  • [Bug 26658] More ongoing changes to make compatible with latest Thundeerbird versions:
    • nsITreeBoxObject was deprecated - causing refresh of folder tree icons to fail
    • document.createElement deprecated - replaced with the new document.createXULelement
    • Improved new HTML color picker behavior when changing background color for inactive tab (live preview was broken)
    • ongoing changes to preferences in dialog system. Lots of moving parts here, so please test rigorously if you are using a "bleeding edge" version of Thunderbird
    • known issue - in Thunderbird 69 the labels of the Tabs vanish when right-clicked. I will fix as soon as I knwo that it's a deprecation cause by XUL changes and not a regression in Thunderbird.
