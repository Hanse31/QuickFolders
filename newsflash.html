Important News Flash!

There is a Bug in the current version of Thunderbird 3.0.1 that causes QuickFolders not to be able to expand for multiple rows anymore; Click here for the bug description and to get updated once it is fixed.

Workaround: insert the following rules into your userChrome.css:

#mail-toolbox { overflow-x:visible !important; overflow-y:visible !important;}
or reinstall the previous version of Thunderbird (3.0). Please note that this is just a workaround and doesn't fix the Thunderbird bug itself.

Since this bug affects ALL extensions that create toolbars, my assumption is that it should be fixed until the next TB release.
